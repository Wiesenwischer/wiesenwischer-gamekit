# 23.1 Crouch-Animationen beschaffen + importieren

> **Commit:** `feat(phase-23): 23.1 Crouch-Animation Assets`
> **Branch:** `feat/crouch-animation-assets`

---

## Ziel

Zwei Crouch-Animationen von Mixamo herunterladen, importieren und korrekt konfigurieren: Crouch Idle und Crouch Walk.

---

## Anweisungen

### 1. Animationen beschaffen

Auf [Mixamo](https://www.mixamo.com/) nach Crouch-Animationen suchen:

**Crouch Idle:**
- **Empfohlen:** "Crouch Idle" oder "Crouching Idle"
- **Format:** FBX for Unity
- **In Place:** Aktivieren
- **Looping:** Ja (loopende Ruhepose)

**Crouch Walk:**
- **Empfohlen:** "Crouch Walk Forward" oder "Sneaking Forward"
- **Alternativen:** "Crouched Walking"
- **Format:** FBX for Unity
- **In Place:** Aktivieren
- **Looping:** Ja (loopende Bewegung)

### 2. Import in Unity

Dateien ablegen unter:
```
Assets/Animations/Locomotion/Anim_CrouchIdle.fbx
Assets/Animations/Locomotion/Anim_CrouchWalk.fbx
```

### 3. Import-Settings konfigurieren (beide Dateien)

**Rig Tab:**
- Animation Type: Humanoid
- Avatar Definition: **Create From This Model**

**Animation Tab (Clips):**
- Loop Time: **Ja** (beide Animationen)
- Root Transform Rotation:
  - Bake Into Pose: **Ja**
  - Based Upon: **Body Orientation**
- Root Transform Position (Y):
  - Bake Into Pose: **Ja**
  - Based Upon: **Feet**
- Root Transform Position (XZ):
  - Bake Into Pose: **Ja**
  - Based Upon: **Original**

> **WICHTIG:** "Bake Into Pose" muss bei ALLEN Root Transform Settings aktiviert sein.

### 4. Animation-Preview

- Beide Animationen in Unity Preview abspielen
- Crouch Idle: Geduckte Ruhepose, kein Drift
- Crouch Walk: Geduckte Laufbewegung, kein Drift
- Loop-Übergänge sind smooth
