# 24.5 Play Mode Verifikation

> **Kein eigener Branch** — manuelle Verifikation im Unity Editor
> **Spec-Referenz:** [FootIK Verbesserungen — Play Mode Verifikation](../../specs/FootIK_Verbesserungen_Spezifikation.md#play-mode-verifikation)

---

## Ziel

Visuelle Verifikation aller drei Fixes im Play Mode. Die Checkliste validiert, dass die Fixes das gewünschte Verhalten zeigen und keine Regressionen einführen.

---

## Vorbereitung

1. Unity Editor öffnen
2. Playground-Szene laden (`Assets/Scenes/Playground.unity`)
3. Play Mode starten
4. Character auf verschiedene Oberflächen bewegen

---

## Checkliste

### Hauptziel: Gebeugte Knie behoben

- [ ] **Flacher Boden, Idle:** Knie sind NICHT mehr gebeugt — Beine stehen gerade wie in der Idle-Animation
- [ ] **Flacher Boden, Crouching:** Keine zusätzliche Knie-Beugung durch IK

### Terrain-Anpassung funktioniert weiterhin

- [ ] **Treppen, Idle:** Füße passen sich weiterhin an die Stufenhöhe an
- [ ] **Slope, Idle:** Fußsohlen folgen der Neigung (Fuß-Rotation)
- [ ] **Kante (ein Fuß hängt):** Body senkt sich korrekt ab

### Übergänge

- [ ] **Übergang flach→Treppe:** Sanfter IK-Übergang (kein abruptes Einrasten)
- [ ] **Übergang Treppe→flach:** Sanfter IK-Übergang zurück

### Keine Regressionen

- [ ] **Walking auf flachem Boden:** Kein visueller Unterschied (IK bereits ausgeblendet)
- [ ] **Running/Sprinting:** Kein visueller Unterschied
- [ ] **Jumping und Landing:** Kein visueller Unterschied
- [ ] **FootIK deaktivieren (Weight=0):** Keine Fehlermeldungen

### Optional: Inspector-Werte prüfen

- [ ] `_maxBodyUpOffset` = 0.05 sichtbar im Inspector
- [ ] `_terrainVarianceThreshold` = 0.03 sichtbar
- [ ] `_footDeadZone` = 0.02 sichtbar
- [ ] Werte können zur Laufzeit geändert werden und wirken sofort

---

## Bekannte Einschränkungen

- Die Terrain-Varianz basiert auf der Höhendifferenz zwischen **beiden Füßen**. Wenn nur ein Fuß einen Raycast-Hit hat, greift die Terrain-Varianz nicht (Weight bleibt bei default 1).
- Die Dead Zone wirkt nur auf die **Positionsdifferenz**, nicht auf die Rotationsdifferenz. Auf leichten Slopes kann es minimal sichtbare Fuß-Rotationen geben, auch wenn die Position kaum abweicht.

---

## Ergebnis

Nach erfolgreicher Verifikation ist Phase 24 abgeschlossen.
