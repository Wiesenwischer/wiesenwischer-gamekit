# 1. Why Traditional Lock-On Fails for BDO-Style Combat

Hard lock-on systems:

-   constrain player movement
-   force camera alignment
-   slow combat flow
-   reduce player freedom

BDO-like combat needs:

-   fast repositioning
-   multiple enemy engagement
-   dynamic direction changes

Therefore:

Targeting must assist --- not control.

------------------------------------------------------------------------

# 2. Soft-Lock Concept

Soft-lock means:

The system helps determine a preferred target without forcing camera
lock.

Player retains:

-   full camera control
-   free movement
-   manual positioning.

------------------------------------------------------------------------

## Soft-Lock Behavior

When attacking:

-   system evaluates nearby targets
-   selects best candidate automatically
-   applies rotation assist (not forced lock).

------------------------------------------------------------------------

# 3. Target Selection Heuristics

Typical evaluation criteria:

## Angle to Camera Center

Targets closer to screen center have higher priority.

Example:

dot(cameraForward, directionToTarget)

------------------------------------------------------------------------

## Distance

Closer targets preferred.

------------------------------------------------------------------------

## Visibility

Prefer targets visible to camera.

------------------------------------------------------------------------

## Combat Context

Examples:

-   already hit target
-   current combo continuation
-   target threat level

------------------------------------------------------------------------

# 4. Target Scoring Example

Each candidate gets a score:

score = angleWeight + distanceWeight + contextWeight

Highest score becomes soft target.

------------------------------------------------------------------------

# 5. Rotation Assist

Instead of locking camera:

Character receives small rotation bias toward selected target.

During active attack phase:

-   increased rotation assist
-   smooth interpolation.

Benefits:

-   player feels precise control
-   no snapping.

------------------------------------------------------------------------

# 6. Camera Independence

Camera must remain independent from target selection.

Camera provides:

-   forward direction
-   screen center reference

Combat system decides:

-   attack direction
-   rotation adjustments.

------------------------------------------------------------------------

# 7. Multi-Target Combat Behavior

BDO-style systems assume multiple enemies.

Soft-lock allows:

-   automatic switching
-   fluid chaining between targets
-   no manual retargeting required.
